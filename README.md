### Start the project:

1. `python -m venv venv`
2. `.\venv\Scripts\activate`
3. `pip install -r requirements.txt`
4. Create `.env` file

5. Run the project:
`flask run --debug`
6. Open website on `http://127.0.0.1:5000`


### .env file:
```
DEBUG=True
ADMIN_NAME=
SECRET_KEY=
SUPABASE_URL=
SUPABASE_KEY=
```
